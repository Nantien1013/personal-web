# 啟動紀錄 — 從零到後端運行

> 紀錄日期：2026-04-23

---

## 一、安裝 MySQL

```bash
sudo apt install mysql-server -y
sudo service mysql start
```

進入 MySQL 設定 root 密碼：

```bash
sudo mysql
```

```sql
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '你的密碼';
FLUSH PRIVILEGES;
EXIT;
```

---

## 二、建立資料庫與資料表

```bash
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS personal_site CHARACTER SET utf8mb4;"
mysql -u root -p personal_site < backend/migrations/001_init.sql
mysql -u root -p personal_site < backend/migrations/002_categories.sql
```

---

## 三、設定 .env

`.env` 必須放在 `backend/` 資料夾內（與 `server.js` 同層）。

```bash
cp .env.example backend/.env
nano backend/.env
```

填入以下內容：

```
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的MySQL密碼
DB_NAME=personal_site
ADMIN_TOKEN=自訂一串字
PORT=3000
ALLOWED_ORIGINS=http://127.0.0.1:5500,http://localhost:5500
```

---

## 四、升級 Node.js（原版本 v12 太舊）

原本 v12 導致 nodemon 出現 `SyntaxError: Unexpected token '?'`。

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt remove --purge nodejs libnode-dev -y
sudo apt autoremove -y
sudo apt install -y nodejs
node -v  # 確認顯示 v20.x.x
```

---

## 五、安裝套件並啟動後端

```bash
cd backend
npm install
npm run dev
```

成功後顯示：`Server running on port 3000`

---

## 六、前端開啟方式

必須用 **Live Server** 開啟，不能直接雙擊 HTML 檔案。

- VS Code 安裝 Live Server 擴充套件
- 右鍵 `index.html` → Open with Live Server
- 網址為 `http://127.0.0.1:5500`

---

## 七、設定 Admin Token（瀏覽器端）

按 `F12` 開啟 DevTools → Console，輸入：

```javascript
localStorage.setItem('adminToken', '你的ADMIN_TOKEN')
```

然後按 `F5` 重新整理，才能新增 / 編輯 / 刪除資料。

---

## 八、遇到的錯誤與解決方式

| 錯誤訊息 | 原因 | 解決方式 |
|---|---|---|
| `node backend/migrations/001_init.sql` 報錯 | `node` 不能執行 SQL 檔 | 改用 `mysql -u root -p < 檔案` |
| `Command 'mysql' not found` | MySQL 未安裝 | `sudo apt install mysql-server -y` |
| `SyntaxError: Unexpected token '?'` | Node.js 版本過舊（v12） | 升級至 Node.js v20 |
| `Failed to fetch` | CORS 被擋 / 後端未啟動 | 用 Live Server 開前端；確認後端有跑 |
| `Access denied (using password: NO)` | `.env` 位置錯誤或密碼未填 | 將 `.env` 放到 `backend/` 內並填入密碼 |
| `未授權：需要管理員 Token` | 瀏覽器沒有 adminToken | 在 Console 執行 `localStorage.setItem` |
| `Column 'volumes_read' cannot be null` | 動漫新增時未傳漫畫欄位預設值 | 後端 `routes/collection.js` 補上預設值 0 |
